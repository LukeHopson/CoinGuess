About

This is a simple coin guessing game where you must guess the country of a coin, given an image of it!

My source code can be found here: GitHub Repository

I made this script for Numista's API to get my data: Numsita-API-Scraper

Image Usage

All images and coin details were sourced from Numista

All images used in my dataset fall under one of the following licenses CC0, CC BY, CC BY-NC, CC BY-NC-SA

Advice for answering:

The following issue cool coins and I've added their own answer:

  • Macau
  • Transnistria
  • Somaliland

Naming conventions:

Use Eswatini for Swaziland

Changes may be made as more coins are added to the game!
